# Box Visualizer

A Python + PySide6 application that visualizes a 3D rectangular box with
adjustable dimensions and keyboard-controlled movement.

## Features
- Adjustable length, width, height
- 3D OpenGL rendering (pyqtgraph)
- WASD + QE movement
- Grid + axis reference

## Requirements
- Python 3.10+
- PySide6
- pyqtgraph
- numpy

## Run
```bash
pip install -r requirements.txt
python src/app.py
